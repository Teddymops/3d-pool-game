A) Integration - top

This game can work as single page in any directory of your site. For example http://yousite.com/game/

You can also integrate the game into the page of your site in the iFrame tag

For example <iframe src="game/" width="640" height="480"></iframe>


B)Customization - top

To configure, open config.js.

It contains the following options:

fullScreen true Enables or disables full screen button. Available values are: true false
